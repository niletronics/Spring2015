All the source code resides in Directory SourceCode
To run the code please follow the steps as below:

For Windows :
Open QuestaSim
Copy Attached NileshDattaniHW1 to a known place and Browse to the same.
Follow the do file named : Windo.do to see results



For Redhat/Linux :
There is a makefile for this environment for this

To run all and see summary (Includes compile, simulate, run and clean) : Default Case
> make all

To compile alone
> make compile

To Simulate and Run : (To be give after compilation)
> make simulate

To clean up work and configuration 
> make compile
